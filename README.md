# 个人博客与简历

> AI 应用开发工程师的个人站点 —— 技术博客、简历与项目展示

## 简介

基于 [Astro](https://astro.build) 构建的静态站点，包含个人简历首页、技术博客和设计系统。

## 技术栈

- **[Astro](https://astro.build)** — 静态站点生成器
- **[Content Collections](https://docs.astro.build/en/guides/content-collections/)** — Markdown 内容管理
- **原生 CSS** — CSS 变量设计系统
- **GitHub Pages** — 静态托管

## 快速开始

```bash
npm install
npm run dev
```

## 部署

推送代码到 `main` 分支，GitHub Actions 会自动构建并部署到 Pages。

在线访问：https://awaygu.github.io/
